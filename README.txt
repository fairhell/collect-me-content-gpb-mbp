
 ==========================================================
     Набор дополнительного содержимого для Коллект!Ми
 ==========================================================
 
 Набор включает в свой состав:
 
  1. Задания (DAG) для генерации данных дашбордов
  2. Соответствующие SQL файлы генерации таблиц и представлений 
  3. Соответствующие CSV файлы с тестовымии данными
  4. Переменные и подключения для заданий

 Для установки запустите скрипт install.sh
 
 В процессе установки производится остановка сервисов Коллект!Ми (и запуск по окончании).

 Перед установкой убедитесь в корректности параметров конфигурации (файл config/install.cfg) -
 параметры сбора почты.
 В дальнейшем параметры можно изменять через веб-интерфейс
